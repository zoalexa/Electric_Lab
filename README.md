# ⚡ ElectricLab — Εκπαιδευτική Προσομοίωση Ηλεκτρολογικών Κυκλωμάτων

## 📌 Περιγραφή

Το **ElectricLab** είναι μια διαδραστική web εφαρμογή που στοχεύει στην εκμάθηση βασικών εννοιών των οικιακών ηλεκτρολογικών κυκλωμάτων.

Απευθύνεται σε:

* Φοιτητές
* Μαθητές
* Εκπαιδευτικούς
* Αρχάριους στον τομέα των ηλεκτρολογικών εγκαταστάσεων

Η εφαρμογή συνδυάζει θεωρία, προσομοίωση και πρακτική εξάσκηση σε ένα ενιαίο περιβάλλον.

---

## 🚀 Λειτουργίες

### 📘 Γλωσσάριο

* Αναζήτηση όρων σε πραγματικό χρόνο
* Αναλυτικές επεξηγήσεις (όχι απλές ορισμοί)
* Παραδείγματα χρήσης
* Οπτικοποίηση με εικόνες

### 🎥 Εκπαιδευτικά Βίντεο

* Οπτική κατανόηση εννοιών
* Παραδείγματα πραγματικών κυκλωμάτων

### 🧠 Σενάρια & Ασκήσεις

* Διαδραστική μάθηση
* Πρακτική εξάσκηση
* Εντοπισμός λαθών σε συνδεσμολογίες

### 💬 Forum

* Ανταλλαγή ιδεών
* Συζήτηση αποριών
* Συνεργατική μάθηση

### ⚠️ Έλεγχος Σφαλμάτων

* Αυτόματη ανίχνευση λανθασμένων συνδέσεων
* Αποφυγή επικίνδυνων καταστάσεων

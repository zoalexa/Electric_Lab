# ⚡ ElectricLab — Εκπαιδευτική Πλατφόρμα Οικιακής Ηλεκτρολογίας

---

## 📌 Περιγραφή

Το **ElectricLab** είναι μια πλήρης διαδραστική web εφαρμογή εκπαίδευσης στις βασικές έννοιες και συνδεσμολογίες οικιακών ηλεκτρολογικών κυκλωμάτων. Λειτουργεί ως **mini LMS (Learning Management System)** εξ ολοκλήρου στον browser, χωρίς backend.

Απευθύνεται σε:
- 👨‍🎓 Φοιτητές & Μαθητές
- 👨‍🏫 Εκπαιδευτικούς
- 🔧 Αρχάριους στον τομέα των ηλεκτρολογικών εγκαταστάσεων

---

## 🚀 Λειτουργίες

### 🔐 Σύστημα Χρηστών & Ρόλων
- **Admin (Καθηγητής):** Πλήρης πρόσβαση σε όλο το υλικό, Admin Panel διαχείρισης μαθητών
- **User (Μαθητής):** Πρόσβαση βάσει προόδου μέσω του Learning Flow
- **Guest:** Πρόσβαση μόνο στο Info — κλειδωμένο υπόλοιπο περιεχόμενο

### 🧭 Learning Flow — Σύστημα Ξεκλειδώματος
Σειριακή ροή μάθησης που αναγκάζει τον μαθητή να ολοκληρώνει κάθε βήμα πριν προχωρήσει:

```
Info → Γλωσσάριο → Βίντεο 1→2→...→9 → Homework + Forum + Υλικό
```

- 🔒 Κλειδωμένα κουμπιά με οπτική ένδειξη (grayscale + opacity)
- 🔓 Toast notifications για κάθε ξεκλείδωμα
- 📊 Progress bar & "Συνέχισε από εκεί που έμεινες"
- 💾 Αποθήκευση προόδου ανά χρήστη μέσω `localStorage`

### 📘 Γλωσσάριο
- Αναζήτηση όρων σε πραγματικό χρόνο με highlight
- Αναλυτικές επεξηγήσεις με παραδείγματα
- Οπτικοποίηση με εικόνες
- Πλοήγηση με πληκτρολόγιο (↑↓)
- Εμφάνιση όλων των όρων μαζί

### 🎥 Εκπαιδευτικά Βίντεο (9 βίντεο)
- Σειριακό ξεκλείδωμα — κάθε βίντεο ξεκλειδώνει το επόμενο
- Κουμπί "Ολοκλήρωση" για παρακολούθηση προόδου
- Progress bar ολοκλήρωσης βίντεο (X/9)
- Grid / List view toggle

### 🧪 Σενάρια Προσομοίωσης (9 σενάρια)
Διαδραστικές προσομοιώσεις βλαβών σε πραγματικά κυκλώματα:

| Σενάριο | Τύπος |
|---------|-------|
| Καμένη Λάμπα | Αντικατάσταση |
| Πρόβλημα Καλωδίωσης | Διαδραστική συνδεσμολογία |
| Αστοχία Λάμπας | Αντικατάσταση |
| Ελαττωματικός Διακόπτης | Αποσύνδεση + Αντικατάσταση |
| Συνδεσμολογία Πρίζας | Διαδραστική συνδεσμολογία |
| Πτώση Αυτόματης Ασφάλειας | Διάγνωση + Επαναφορά |
| Βραχυκύκλωμα | Απομόνωση βλάβης |
| Απουσία Γείωσης | Προσομοίωση κινδύνου |
| Υπερθέρμανση Πρίζας | Πολλαπλά βήματα επίλυσης |

- Σειριακό ξεκλείδωμα σεναρίων
- Κουμπί "Ολοκλήρωση Σεναρίου" μετά από επιτυχή αλληλεπίδραση

### 📝 Ασκησιολόγιο
**Βασικές & Προχωρημένες Ασκήσεις** με:
- **Practice mode:** Άμεση ανατροφοδότηση, δεν προχωράει αν είναι λάθος
- **Exam mode:** Χρονόμετρο, shuffle ερωτήσεων/απαντήσεων, 100% για ξεκλείδωμα
- Τύποι ερωτήσεων: MCQ, Σωστό/Λάθος, Συμπλήρωση κενών, Αντιστοίχιση
- Review mode με επεξηγήσεις (💡 Εξήγηση) ανά ερώτηση
- Progress bar ολοκλήρωσης ασκησιολογίου

### 🎯 Quiz (5 ενότητες)
| Ενότητα | Θέμα | Ερωτήσεις |
|---------|------|-----------|
| 1 | Εισαγωγή | 15 |
| 2 | Κυκλώματα | 15 |
| 3 | Συνδεσμολογίες | 15 |
| 4 | Ασφάλεια & Βλάβες | 15 |
| 5 | Τυχαίο Mix | 40 |

- Σειριακό ξεκλείδωμα ενοτήτων
- Review mode με feedbacks
- 100% στο exam για ξεκλείδωμα επόμενης ενότητας

### 💬 Forum (4 κατηγορίες)
- Ερωτήσεις Μαθητών
- Σενάρια Βλαβών
- Ασκήσεις & Homework
- Ιδέες & Προτάσεις

Λειτουργίες:
- Δημιουργία, επεξεργασία, διαγραφή thread
- Απαντήσεις με δικαιώματα ανά ρόλο
- Timestamps δημιουργίας
- Αριθμός αναρτήσεων ανά κατηγορία

### ⚙️ Admin Panel
- Προβολή προόδου κάθε μαθητή (% + checklist βίντεο + ξεκλειδωμένα)
- Summary bar (σύνολο χρηστών, ολοκλήρωσαν, σε εξέλιξη, μέση πρόοδος)
- Reset προόδου ανά χρήστη
- Unlock all για άμεση πρόσβαση μαθητή
